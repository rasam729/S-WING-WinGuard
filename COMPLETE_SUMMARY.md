## 🚀 How to Use Everything

### Quickest Way (1 Minute)

```
1. Double-click: start-all.bat
2. Wait for services to start
3. Browser opens automatically
4. Done! ✅
```

### Mobile App (10 Minutes)

```bash
cd apps/citizen-app
npm install @capacitor/android
npx cap add android
npm run build
npx cap copy
npx cap open android
# Click Run in Android Studio
```

### Offline Mode (5 Minutes)

```bash
cd apps/citizen-app
npm install dexie workbox-webpack-plugin @capacitor/network
# Copy code from OFFLINE_SETUP.md
```
